# Curso HTML, CSS y JavaScript - Ejercicios Prácticos

Este repositorio contiene ejercicios prácticos del bootcamp nodejs de Devlights. El objetivo es practicar conceptos fundamentales de JavaScript a través de ejercicios organizados por clases.

## Descripción del Proyecto

El proyecto está estructurado para ir acumulando ejercicios de todas las clases del curso. Actualmente incluye:

- **script.js**: Ejercicios básicos de JavaScript (bucles, condicionales, arrays, matrices, métodos de array como reduce, some, every)
- **Clases 1 y 2/ejercicios.js**: Ejercicios avanzados trabajando con arrays de objetos (profesionales), utilizando métodos funcionales como filter, map, find, reduce, some y every
- **Clases 1 y 2/profesionales.js**: Datos de ejemplo con información de profesionales para los ejercicios

## Estructura del Proyecto

```
/
├── index.html              # Página principal HTML
├── script.js               # Ejercicios básicos de JavaScript
├── package.json            # Configuración del proyecto Node.js
├── Clases 1 y 2/
│   ├── ejercicios.js       # Ejercicios con arrays de objetos
│   └── profesionales.js    # Datos de profesionales para ejercicios
└── README.md               # Este archivo
```

## Tecnologías Utilizadas

- **JavaScript (ES6+)**: Lenguaje principal para los ejercicios
- **Node.js**: Para ejecutar los scripts desde la línea de comandos
- **HTML5**: Estructura básica de la página web

## Cómo Ejecutar los Ejercicios

### Opción 1: Desde el navegador

1. Abre `index.html` en tu navegador web
2. Los ejercicios se ejecutarán automáticamente y podrás ver los resultados en la consola del navegador (F12 → Console)

### Opción 2: Desde Node.js

1. Asegúrate de tener Node.js instalado
2. Ejecuta el comando:
   ```bash
   npm start
   ```
   Esto ejecutará `node "./Clases 1 y 2/ejercicios.js"` y mostrará los resultados en la terminal

## Contenido de los Ejercicios

### script.js

- Bucles: for, while, do-while
- Estructuras de control: if/else, operador ternario, switch
- Arrays y matrices
- Métodos de array: reduce, some, every

### Clases 1 y 2/ejercicios.js

- Filtrado de arrays (filter)
- Transformación de datos (map)
- Búsqueda de elementos (find)
- Reducción de arrays (reduce)
- Validaciones con some y every
- Trabajo con objetos y propiedades
